Belajar HTML & CSS

Apa itu HTML?

HTML (HyperText Markup Language) digunakan untuk membangun struktur dasar sebuah halaman web.

Contoh Kode HTML:

<!DOCTYPE html>
<html>
  <head>
    <title>Belajar HTML</title>
  </head>
  <body>
    <h1>Halo Dunia!</h1>
    <p>Ini paragraf pertama saya.</p>
  </body>
</html>

Apa itu CSS?

CSS (Cascading Style Sheets) digunakan untuk mempercantik dan mengatur tampilan halaman web.

Contoh Kode CSS:

body {
  background-color: #f4f4f9;
  color: #333;
  font-family: Arial, sans-serif;
}

h1 {
  color: #0077cc;
  text-align: center;
}
